Statistical Analysis – C1M1 Assessment

Author: Marcos Sgorla
Date: February 2026

Overview

This project implements basic statistical analysis on a fixed dataset of 40 unsigned 8-bit integers. The program computes and prints:

Minimum

Maximum

Mean

Median

The dataset is sorted in descending order (largest to smallest) prior to median calculation.

Files

stats.c – Function implementations

stats.h – Function declarations and documentation

README.md – Project description

Build & Run
gcc stats.c -o stats
./stats

Purpose

Demonstrates:

Modular C design

Header/source separation

Pointer-based array processing

Basic sorting and statistical computation
